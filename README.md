Hello! 👋
This repository contains the materials for the "Python Basics for Data Analysts" training, hosted by Marco and Gabriella. Our goal isn't to teach Python from scratch like computer scientists, but rather to show you how to use it as a practical tool for everyday data analysis.

🎯 Course Objective
The course is structured around real-world use cases that we tackle daily in our work, pairing theory directly with hands-on practice.

🛠 What's inside this repository
Every notebook is designed to be interactive and practically oriented:
- Practical day-to-day examples: Code snippets and patterns we actually use to extract, clean, and manipulate data.
- Applied theory: Brief conceptual explanations immediately followed by their Python implementation.
- Dummy Data included: Every script comes with pre-loaded mock datasets. You'll be able to run the code, tweak it, and get your hands dirty without having to connect to external databases or hunt for test data.

⚠️ What we WON'T cover (Prerequisites)
Since this course is tailored for analysts, we will not start from the absolute basics of programming. We will assume you are already familiar with some general concepts (often known to those who use SQL or other languages), including:
- What variables and assignments are.
- Fundamental data types (integers, strings, booleans).
- The basic concept of functions.
- Standard conditional logic (conceptual IF/ELSE).

### 📚 Recommended Prerequisites Resources
If you feel rusty on the basics, check out these resources before the training:
- [DataCamp: Intro to Python for Data Science](https://www.datacamp.com/courses/intro-to-python-for-data-science) - Great interactive exercises.
- [LearnPython.org](https://www.learnpython.org) - Quick and free interactive tutorials.

🚀 How to use this material during the lesson
No installation needed: Just click the Open in Colab badge at the top of the page (or next to each notebook).
Make a copy: Colab will open the cloud environment. Remember to save a copy to your Google Drive (from the menu File > Save a copy in Drive) if you want to keep your edits and notes.
Run and experiment: During the lesson, follow along with our explanations and try running the code cells (by pressing Shift + Enter). Change the parameters, break the code, and figure out how to fix it with us!
